Aula

Variáveis:
- var: primeiro jeito de declarar uma variável.

- let: recebe novos valores e tem escopo definido.

- const: valores fixos, que não mudam no programa.

Interações com o usuário:
- alert("Mensagem"): Exibe uma mensagem com botão "OK".

- prompt("Pergunta"): Permite entrada de dados pelo usuário.

- confirm("Pergunta"): Mostra opções "OK" e "Cancelar".

Exemplo de cálculo de média:
- O usuário insere duas notas.

- A média é calculada com (nota1 + nota2) / 2.

- Se for maior ou igual a 6, o aluno é aprovado; caso contrário, reprovado.

Estruturas de repetição:
- for: laço com quantidade fixa de repetições.

- while: repete enquanto a condição for verdadeira.

- do while: executa pelo menos uma vez antes de verificar a condição.
